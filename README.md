# 图片压缩工具

一个简单易用的在线图片压缩工具，采用 Material Design 风格设计。

## 功能特点

- 支持上传 PNG、JPG 格式图片
- 支持自定义压缩比例
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小
- 支持下载压缩后的图片
- 响应式设计，支持各种设备

## 技术实现

- 使用原生 HTML5、CSS 和 JavaScript 开发
- 使用 HTML5 的 File API 处理文件上传和下载
- 使用 Canvas API 进行图片压缩
- 采用 Flexbox 布局实现响应式设计
- 使用 Material Design 样式组件
